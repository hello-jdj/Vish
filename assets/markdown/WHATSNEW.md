## Notes

### v1.1.1
New theme system ! You can now add your own themes by placing .yml files in the "themes" folder in your config directory. 
Custom themes can be shared with others, and you can even create your own themes to share with the community ! Check [the repository](https://github.com/Lluciocc/vish-theme-collection) for more details on how to create themes.

### v1.1.0
New comments are out ! You can now mode nodes by group, changing the accent color, and more !

### v1.0.0
Vish is in Beta. Expect bugs and missing features. Stay tuned for updates!
You can find the latest updates in the repository: [Vish on GitHub](https://github.com/Lluciocc/vish)
